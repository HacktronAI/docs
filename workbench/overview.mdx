Hacktron Workbench combines the Hacktron CLI with IDE extensions for VS Code, Cursor, and Windsurf.

Use the CLI for terminal-native investigations and the IDE extension for editor-native workflows.

## Agent Packs

Agent packs are collections of agents for specific technologies or types of vulnerabilities.

Hacktron's community agents are available to all users and are continually updated to keep up to date with the latest vulnerabilities and security research.

<AccordionGroup>
  <Accordion title="Community agents" icon="globe">
    We continually perform security research, finding zero-day vulnerabilities in products like **ChatGPT Atlas, Perplexity's Comet Browser, Windsurf, and Lovable**.

    Beyond standard vulnerability patterns, we synthesise our novel findings into agent packs that can be used to find similar vulnerabilities in your codebase.

    This ensures that Hacktron can find **variants of real vulnerabilities that top engineering teams have missed**.
  </Accordion>

    <Accordion title="Personal agents" icon="user">
    Hacktron **learns from collaborating with you** and automatically creates personal agents customised to your codebase.

    These agents are dynamically created when you use Hacktron, and are **private to you**.
  </Accordion>
</AccordionGroup>

Learn more about agent packs [here](/workbench/cli/agents).
