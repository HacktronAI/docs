## What is included in Hacktron Workbench?

Hacktron Workbench includes:

- **Hacktron CLI** for terminal-native security workflows
- **Hacktron IDE extension** for in-editor security workflows

Use both tools with the same Hacktron account.

## Install the extension

<Steps>
  <Step title="Open your IDE marketplace">
    Open the extension marketplace in **VS Code**, **Cursor**, or **Windsurf**.
  </Step>
  <Step title="Search for Hacktron">
    Search for `Hacktron` and install the extension published by `HacktronAI`.

    You can also install directly from the VS Code marketplace:
    [Hacktron extension](https://marketplace.visualstudio.com/items?itemName=HacktronAI.hacktron)
  </Step>
  <Step title="Sign in">
    Open the Hacktron extension panel and sign in with your Hacktron account.
  </Step>
</Steps>

## Verify your setup

After signing in, confirm:

- The extension shows your account as connected
- You can start a new Hacktron session from the IDE
- You can run a simple prompt, such as `Check for XSS vulnerabilities in this file`

## Features

The IDE extension supports:

- **Background sessions** to run workflows in parallel
- **Bring Your Own Keys (BYOK)** for custom model providers
- **Personal agents** configured from settings
